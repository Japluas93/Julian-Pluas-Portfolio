Daily Planner

HTML | CSS | Javascript | jQuery | Client-Side Storage

A simple daily planner application that I designed to help anyone manage their busy schedule. I used Bootstrap because I wanted to get more practice using the row and columns syntax. The application is for the most part responsive and it also has working client side storage functionality. Whenever the user inputs some text in the input area and presses the save button that data will remain in the input area, even after you refresh the page.
